Tạo Supabase miễn phí (5 phút):
1. Vào supabase.com → Start your project
2. Đăng nhập bằng GitHub hoặc Google
3. Nhấn New project → đặt tên → chọn region Southeast Asia
4. Vào Project Settings → API → copy Project URL và anon public key
5. Vào SQL Editor → chạy SQL sau để tạo bảng:
CREATE TABLE vocabulary ( id bigint generated always as identity primary key, word text not null, meaning text, created_at timestamptz default now() ); ALTER TABLE vocabulary ENABLE ROW LEVEL SECURITY; CREATE POLICY "Allow all" ON vocabulary FOR ALL USING (true) WITH CHECK (true);
